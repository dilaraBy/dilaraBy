## 🚀 Featured Projects

### [Student Mental Health Dashboard](https://github.com/dilaraBy/student-mental-health-dashboard)
Streamlit analytics platform exploring mental health patterns among Bangladeshi university students. Layered architecture (services / repository / utils), 12 test modules with TDD, interactive choropleth maps, full CRUD over SQLite.
**Stack:** Python · Streamlit · SQLite · Plotly · pytest

### [Customer Purchase Pattern Analysis](https://github.com/dilaraBy/customer-purchasing-patterns)
Graph-based market basket analysis on 14,963 supermarket transactions. Custom `WeightedUndirectedGraph` from scratch, BFS / top-K queries, and a CLI for interactive exploration. 95 tests, CI on three Python versions.
**Stack:** Python · NetworkX · Matplotlib · pytest

### [Diabetes Risk Prediction](https://github.com/dilaraBy/diabetes-risk-prediction)
End-to-end CDC BRFSS classification pipeline: data prep → XGBoost with Platt calibration → SHAP explainability → threshold optimization for recall-oriented screening.
**Stack:** Python · XGBoost · scikit-learn · SHAP

### [EEG Mental Workload Classification](https://github.com/dilaraBy/eeg-workload-classification)
Deep-learning pipeline for EEG-based cognitive workload classification. CNN-LSTM baseline, BiLSTM + temporal attention, Mixup / DANN / SWA improvements, and fine-tuning of the CBraMod foundation model.
**Stack:** Python · PyTorch · NumPy · MNE
